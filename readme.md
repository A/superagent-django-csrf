# superagent-django-csrf

Patch to add `csrftoken` from cookies as `X-CSRFToken` header to every superagent's request.

## Usage

```js
// Just require patch
require('superagent-django-csrf');
```

## Install

```shell
npm install superagent-django-csrf
```
